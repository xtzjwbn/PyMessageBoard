# PyMessageBoard

## Command ##
```
login <username> <password> - 登录账号
logon <username> <password> <nickname> - 创建账号
logoff - 退出登录
list <n> - 查看留言版，n表示查看第几页。如果省略n显示第一页。
add_msg <content> - 增加留言
del_msg <uuid> - 删除留言
like_msg <uuid> - 点赞留言
unlike_msg <uuid> - 取消点赞
```
